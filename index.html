Абзаци тексту

HTML, або HyperText Markup Language, є основною мовою розмітки для створення веб-сторінок.

За допомогою HTML можна створювати структуровані документи, що містять заголовки, абзаци, списки та багато іншого.

HTML підтримує різні стилі форматування тексту, такі як курсив, жирний та підкреслений текст.

Додаткові стилі

Код можна відображати за допомогою тега <code>.

Також можна використовувати тег <samp> для відображення зразків виведення.

Тег <kbd> використовується для позначення клавіш, наприклад, Ctrl + C.
